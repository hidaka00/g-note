# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-15

### Added
- Initial release
- Global folder registration and management
- Sidebar TreeView for registered folders
- File operations: create, rename, delete
- Folder operations: create, rename, delete
- Context menu with node-type specific actions
- Open file in editor
- Reveal in OS file explorer
- Manual refresh
- Persistence via globalState
- File name validation (Windows reserved characters)
- Collision detection for file/folder creation
- Delete confirmation dialog
- Output channel logging
